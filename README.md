# Dashboard UI

A sample dashboard UI project developed in VueJS 3.

## Technology Stack

- Typescript
- VueJS 3
- Node (v14.x.x LTS)

## Project setup

```
pnpm install
```

### Compiles and hot-reloads for development

```
pnpm run serve
```

### Compiles and minifies for production

```
pnpm run build
```

### Lints and fixes files

```
pnpm run lint
```
